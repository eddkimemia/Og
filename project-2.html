PROJECT: SHIFTBOARD

$ cat tech_loadout.txt
┌────────────┬────────────────────────────────────┐ │ Frontend │ React, Tailwind CSS │ │ Backend │ Node.js, Express │ │ Database │ PostgreSQL │ │ Infra │ Africa's Talking API, date-fns │ └────────────┴────────────────────────────────────┘

THE MISSION

Restaurant and hotel managers were wasting hours every week on paper-based staff rotas. ShiftBoard provides a digital drag-and-drop calendar for scheduling.

THE APPROACH

I developed a real-time calendar interface using CSS Grid and integrated Africa's Talking API to send automated WhatsApp notifications to staff whenever a shift was assigned or changed.

THE CODE

const notifyStaff = async (staffMember, shift) => {
  const message = `Habari ${staffMember.name}, you have a new shift on ${shift.date} at ${shift.time}`;
  await AT.sms.send({
    to: staffMember.phone,
    message: message
  });
};

THE OBSTACLES

Optimizing calendar performance for mobile browsers. I switched from a heavy third-party library to a custom-built lightweight grid solution, reducing initial load time by 60%.

THE RESULTS

Cut manager scheduling time by 5 hours per week and improved staff punctuality by 25% for our pilot clients in Nairobi.
