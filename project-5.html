PROJECT: LEARNDESK

$ cat tech_loadout.txt
┌────────────┬────────────────────────────────────┐ │ Frontend │ React │ │ Backend │ Node.js, Express │ │ Database │ PostgreSQL │ │ Infra │ Cloudinary, M-Pesa, Stripe │ └────────────┴────────────────────────────────────┘

THE MISSION

Independent educators in Kenya lacked a platform to sell video courses with local payment options like M-Pesa, forcing them to use international platforms with high fees.

THE APPROACH

I built a lightweight LMS that supports M-Pesa for local students and Stripe for international ones. Video lessons are securely hosted on Cloudinary with access control.

THE CODE

router.get('/lessons/:id', authenticateToken, async (req, res) => {
  const enrollment = await db.Enrollment.findOne({
    where: { userId: req.user.id, courseId: lesson.courseId }
  });
  if (!enrollment) return res.sendStatus(403);
  res.json({ videoUrl: lesson.cloudinaryUrl });
});

THE OBSTACLES

Securing video content from unauthorized downloads. I implemented signed URLs and expiring tokens to ensure only enrolled students can access the lessons.

THE RESULTS

Launched with 5 independent educators hosting 12 courses. Educators retain 90% of their revenue compared to 50-70% on international platforms.
