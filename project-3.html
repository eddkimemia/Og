PROJECT: FORMCRAFT

THE MISSION

Local businesses needed a low-cost alternative to Typeform that could be easily embedded in their existing websites without technical expertise.

THE APPROACH

I built a no-code drag-and-drop form builder with a lightweight vanilla JS runtime for the embedded forms. This ensures minimal impact on the host site's performance.

THE CODE

(function() {
  const script = document.currentScript;
  const formId = script.getAttribute('data-form-id');
  const container = document.createElement('div');
  container.id = `formcraft-${formId}`;
  script.parentNode.insertBefore(container, script);
  loadForm(formId, container);
})();

THE OBSTACLES

Preventing host site CSS from bleeding into the embedded forms. I utilized the Shadow DOM to ensure complete style isolation for every FormCraft embed.

THE RESULTS

Successfully launched and currently powers lead generation forms for over 15 startups in the Nairobi tech ecosystem.
