API calls that retry on failure with exponential delay.

THE CODE

async function retry(
  fn: () => Promise,
  retries = 3,
  delay = 1000
): Promise {
  try {
    return await fn();
  } catch (error) {
    if (retries === 0) throw error;
    await new Promise(r => setTimeout(r, delay));
    return retry(fn, retries - 1, delay * 2);
  }
}

HOW IT WORKS

Loops through retries and uses setTimeout wrapped in a Promise.

REAL-WORLD EXAMPLE

await retry(() => fetch("/api"));

COMMON PITFALLS

Be careful not to retry on 404 errors which will never succeed.
