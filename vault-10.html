SNIPPET: memoize-with-ttl.ts

$ cat use_case.txt
Cache results with expiration.

THE CODE

function memoize any>(
  fn: T,
  ttl: number
): T {
  const cache = new Map();
  return ((...args: any[]) => {
    const key = JSON.stringify(args);
    const cached = cache.get(key);
    if (cached && cached.expiry > Date.now()) return cached.value;
    const result = fn(...args);
    cache.set(key, { value: result, expiry: Date.now() + ttl });
    return result;
  }) as any;
}

HOW IT WORKS

Stores results in a Map alongside a timestamp.

REAL-WORLD EXAMPLE

const cachedFetch = memoize(fetchUser, 60000);

COMMON PITFALLS

Map can grow infinitely if keys are not managed.
