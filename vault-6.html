Type-safe Array.groupBy polyfill.

THE CODE

function groupBy(
  arr: T[],
  fn: (item: T) => string | number
): Record {
  return arr.reduce((acc, item) => {
    const key = fn(item);
    if (!acc[key]) acc[key] = [];
    acc[key].push(item);
    return acc;
  }, {} as Record);
}

HOW IT WORKS

Uses Array.reduce to build an object of groups.

REAL-WORLD EXAMPLE

groupBy(users, "role");

COMMON PITFALLS

Keys must be strings or numbers to work as object properties.
